About this website

This website is homework for Web Integrated Design Practice Course, whose server side uses Python 3.10.13 and Django 5.0.3.

Source code of this website is served on Github.

Deployment

To deploy this website yourself, you need to install Python at 3.10 or above, and install Django using pip.

In addition, Django's sslserver app is also needed if you want to run server on https. If not, remember to delete sslserver in INSTALLED_APP in wiki/settings.py.

Before launching the server for the first time, you should execute makemigrations and migrate in order to make database function properly.
